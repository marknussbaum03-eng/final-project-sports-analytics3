# Final Project: NBA Team Performance Analysis (2019-20 Season)

## Project Overview
Analyzing Charlotte Hornets, New Orleans Pelicans, and LA Clippers performance during the 2019-20 COVID-shortened NBA season using `nba_api` and pandas.

## Key Analysis
- Fetched 194+ games via `LeagueGameFinder`
- Points per game, win/loss trends, matchup analysis
- Potential: Advanced metrics, visualizations, player impact

## Files
- `hornets_pelicans_clippers_2019_20_games.csv` - Raw game data
- `nba_analysis.ipynb` - Jupyter notebook (coming soon)

## Tech Stack
- Python, pandas, nba_api, matplotlib/seaborn
- GitHub for version control
